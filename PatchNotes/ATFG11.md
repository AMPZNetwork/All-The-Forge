## Changed/Fixed
- We've added a dedicated **Mod Button** to the pause menu, making it easier to view your currently installed mods while being in-game.
- The **"Report a bug"** button is now fully enabled and functional.
- Fixed inverted filter behaviour specifically concerning unlisted resource types.
- Resolved an issue where creative chemical storages could quickly lead to overflow crashes.
- Level 3 keycards can now be crafted correctly.
- Patched a few duplication exploits.
- Create tracks now stack properly in the inventory.
- Fixed a critical bug where using an export card with a fuzzy card could sometimes delete items.
- The **"Open to LAN"** button now matches the consistent style of the rest of the pause menu.
- Updated the BisectHosting promotion logic for quicker detection of All The Forge v11 during checkout.
- Removed a few leftover assets from early development builds.
- Other known issues have been addressed.
